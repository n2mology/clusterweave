# Fungi-only example

This retained public example contains 50 fungal assemblies and the fungal
ClusterWeave output surface. `accessions.txt` is the reproducible input list;
`accessions_fungusID_taxonomyID.txt` records the generated identifiers and NCBI
taxonomy lineage. Curated outputs live under `figures/` and `summary/`.

The accession list is locked for v1.0.0. The list and its derived outputs must
remain unchanged; a future correction should use a new version and recorded
provenance.

Canonical figure names include `fungi_big_scape_multipanel.svg` and
`bgc_overlap.svg`. No raw genomes, raw third-party result trees, private job
metadata, tokens, logs, or machine paths are included in this repository-safe
example snapshot. A completed authenticated run has a different purpose:
**Download package** preserves the staged genome and BGC GenBank files needed
for further review of that run.
