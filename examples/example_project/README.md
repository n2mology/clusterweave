# Example Project Assets

This folder is reserved for public-safe example assets that help demonstrate `ClusterWeave` without exposing private or redistribution-restricted project material.

Add files here only after provenance and redistribution review.

Current public-safe example bundle:

- `clusterweave_smoke_derived_outputs/`: compact derived outputs from the
  smoke run, excluding genome/sequence files and third-party working trees.

Unlike `tests/`, this directory is meant for human-facing example material such as:

- tiny walkthrough input bundles
- public-safe example outputs or screenshots
- manuscript/release companion files
