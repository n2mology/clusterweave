# ClusterWeave Smoke Derived Outputs

This directory is the recommended public example bundle from the
`clusterweave_smoke` run.

It intentionally shares only compact, derived outputs that demonstrate the
canonical workflow without republishing genome assemblies, BGC GenBank records,
protein translations, staged clinker inputs, or full third-party result trees.

## Included

- `summary/all_tools_shared_unshared_summary.csv`
  - aggregate BGC/GCF counts by genome, tool, entity type, and class
- `summary/family_atlas_shortlist.md`
  - human-readable dataset-wide family-atlas shortlist with conservative
    interpretation text
- `summary/family_atlas_shortlist.tsv`
  - tabular form of the same atlas shortlist for reproducible examples
- `summary/ecofun_metadata_normalized.tsv`
  - accession/genome metadata scaffold with intentionally blank ecology fields
    so readers can see how optional ecology labels can be added
- `figures/bgc_calls_by_tool_category.svg`
  - rendered per-genome, per-tool summary plot from the aggregate table

## Excluded

The following smoke-run outputs are deliberately not included here:

- genome assemblies and annotations: `.fna`, `.fa`, `.fasta`, `.gb`, `.gbk`,
  `.gbff`, `.gff`, `.gff3`, `.faa`
- antiSMASH, FunBGCeX, funannotate, BRAKER3, and BiG-SCAPE working directories
- staged clinker panel inputs and generated panel HTML
- downloaded third-party resources, containers, and caches
- large logs and temporary working files

These excluded files can be regenerated by users from the accession list and
documented workflow commands when they have the appropriate local tools and
resources.

## Sharing Rationale

The atlas shortlist is the best single example output because it shows the
workflow's practical review surface: family context, consensus support,
annotation hints, ranking, and conservative safe-claim text. The aggregate
summary table and figures show how the workflow can be summarized without
exposing sequence data. The blank ecology metadata file is included on purpose
as a template-like example for users who want to enable ecology-aware ranking.
