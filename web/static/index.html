Third-party tools and citations ClusterWeave is a portal to public, open-access biosynthetic discovery tools.

We are grateful to the maintainers of these tools and reference datasets. ClusterWeave depends on their public availability and wraps them into a guided workflow so non-coding domain experts can participate in BGC target discovery. Please cite the upstream tools as well as ClusterWeave when outputs support your work.

antiSMASHAGPL
Containerized BGC detection with `antismash/standalone:8.0.4`.
Blin et al. antiSMASH 8.0. Nucleic Acids Research 2025. doi:10.1093/nar/gkaf334.
FunBGCeXMIT
Fungal BGC extraction through the ClusterWeave-built FunBGCeX container.
Tang and Matsuda, Nat. Commun. 15, 4312 (2024), doi:10.1038/s41467-024-48771-7; Chan et al., JACS 147, 42100-42109 (2025), doi:10.1021/jacs.5c16272.
funannotateBSD-2
Permissive annotation fallback for compatible fungal genome inputs.
Palmer and Stajich. Funannotate v1.8.1: Eukaryotic genome annotation. Zenodo 2020. doi:10.5281/zenodo.1134477.
BiG-SCAPEAGPL
Family graph context via `ghcr.io/medema-group/big-scape:2.0.0-beta.6`.
BiG-SCAPE 2.0 and BiG-SLiCE 2.0, Nat. Commun. 2026. doi:10.1038/s41467-026-68733-5; Navarro-Munoz et al., Nat. Chem. Biol. 2020. doi:10.1038/s41589-019-0400-9.
clinkerMIT
Synteny comparison panels via the BioContainers clinker image.
Gilchrist and Chooi. clinker and clustermap.js. Bioinformatics 2021. doi:10.1093/bioinformatics/btab007.
MIBiGCC BY 4.0
Versioned reference BGC context used by upstream tools and optional local caches.
Zdouc et al. MIBiG 4.0. Nucleic Acids Research 2025. doi:10.1093/nar/gkae1115.
PfamCC0
Protein family HMM reference data used in family-context stages.
Paysan-Lafosse et al. Pfam protein families database. Nucleic Acids Research 2024. doi:10.1093/nar/gkae997.

This list is a processed draft and will grow as more public tools are added to ClusterWeave. The hosted portal avoids restricted optional annotation fallbacks; local command-line users should review `THIRD_PARTY.md` before enabling extra tooling.
